# Mermaid Studio Local Examples

Use these reference notes as quick local examples while authoring Mermaid.

Available example topics:
- flowchart
- sequence
- class
- state
- er
- gantt
- mindmap
- gitgraph
- architecture

You can open these from Mermaid Studio hover help via **Open local examples**.

Other quick reference paths:
- Type `/snippet` in Mermaid editors to browse insertable starters/micro snippets
- Use **Mermaid Studio: Open Reference for Symbol Under Cursor** for local examples or official docs
- Use **Ctrl/Cmd + Click** on supported Mermaid keywords/operators to jump to the bundled local examples
